262/* CPU States
263user – time spent in user mode.
264nice – time spent in user mode with low priority.
265system – time spent in system mode.
266idle – time spent in the idle task.
267iowait – time waiting for I/O to complete.
268irq – time servicing hardware interrupts.
269softirq – time servicing software interrupts.
270steal – time spent in other operating systems when running in a virtualized environment.
271guest – time spent running a virtual CPU for guest operating systems.
272guest_nice – time spent running a low priority virtual CPU for guest operating systems.
